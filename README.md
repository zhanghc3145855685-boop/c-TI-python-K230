# c-TI-python-K230

本项目基于 **2025年全国大学生电子设计竞赛（电赛）E题** 规范开发，实现了一套基于边缘视觉引导的运动控制系统。

系统采用“双核/双主控”架构：利用 K230 边缘计算芯片进行目标检测与坐标解算，并通过异步串行通信将位置偏差实时同步至 TI 微控制器（MSPM0 系列），由底层驱动电路完成云台追踪与底盘运动的闭环控制。

因为项目时间过于久远，当时不注重文件管理，我甚至不知道上传的是不是最终版代码，也是认识到版本管理的重要性了，代码只有基础部分。
